# Pot 桌面翻译软件 – 功能模块说明（面向用户）

本说明从「产品/使用」角度介绍 Pot 的主要功能模块，方便你对照界面快速上手。

## 1. 常驻托盘与启动行为

- 常驻系统托盘（右下角或菜单栏）
  - 关闭所有窗口后，程序不会完全退出，而是最小化到托盘，保证快捷键和剪贴板监控等功能继续工作。
  - 你可以通过托盘图标右键菜单快速打开：翻译窗口、识别窗口、设置、关于、检查更新等。
- 开机自启（可选）
  - 是否随系统启动，可以在「设置 → 通用」中打开或关闭。

## 2. 快速划词翻译

- 功能说明
  - 在任意应用中选中一段文字，按下你设置的「划词翻译」快捷键，Pot 会自动弹出一个小的翻译窗口，并显示翻译结果。
- 使用步骤
  - 第一步：在「设置 → 快捷键」中给「划词翻译」设置一个全球快捷键，例如 `Ctrl+Shift+E`。
  - 第二步：在浏览器、PDF、Word、代码编辑器等任意软件中，选中要翻译的文字。
  - 第三步：按下刚才设置的快捷键，Pot 翻译窗口会在光标附近弹出并自动填充选中的文本。
- 适用场景
  - 阅读外文网页/文档时随手查整句。
  - 阅读技术文档或论文时，快速理解长句含义。

## 3. 输入翻译窗口（手动输入）

- 功能说明
  - 除了划词翻译，你也可以主动打开一个专门的翻译窗口，手动输入或粘贴内容进行翻译。
- 打开方式
  - 在托盘菜单中选择「输入翻译」或类似入口。
  - 或者在「设置 → 快捷键」中为「输入翻译」设置全局快捷键，随时呼出。
- 功能特点
  - 支持多段文字翻译。
  - 支持切换源语言/目标语言。
  - 可搭配多个翻译服务同时显示结果，方便对比。

## 4. 截图翻译 / OCR 识别

### 4.1 截图识别（仅提取文字）

- 功能说明
  - 对屏幕上的任意区域进行截图，Pot 会对截图内容进行文字识别（OCR），并在识别窗口中显示结果。
- 使用步骤
  - 在「设置 → 快捷键」中为「截图识别」设置一个快捷键。
  - 按下快捷键后，屏幕会进入截图模式：
    - 在 macOS 上，会调用系统截图工具。
    - 在 Windows / Linux 上，会弹出 Pot 的全屏截图界面。
  - 用鼠标框选需要识别的区域，完成后自动打开识别窗口，展示识别出的文本。
- 适用场景
  - 识别图片中的文字。
  - 提取 PDF/图片/视频中的字幕或内容。

### 4.2 截图翻译（识别后立即翻译）

- 功能说明
  - 在截图后，不仅识别文字，还自动将识别出的内容丢到翻译窗口进行翻译。
- 使用步骤
  - 在「设置 → 快捷键」中为「截图翻译」设置一个快捷键。
  - 截图完成后，Pot 会自动打开翻译窗口并显示翻译结果。
- 适用场景
  - 不只想要原文，还希望直接看到翻译。

## 5. 翻译窗口 – 多引擎对照翻译

翻译窗口是 Pot 的核心主界面，对应「Translate」窗口。

- 上方：源文本输入区
  - 可以手动输入、粘贴文本，或者通过划词/截图等方式自动填入。
  - 支持清空、复制等基本操作。
- 中间：语言栏
  - 显示「源语言」和「目标语言」。
  - 支持手动选择语言（例如从英文翻译到中文），也支持自动检测（取决于启用的服务）。
- 下方：多个翻译卡片
  - 每个卡片代表一个翻译服务（例如：DeepL、百度、有道、Google、Bing 等）。
  - 同一段文本会并行发给多个服务，方便你对比不同服务的翻译结果。
  - 可以通过拖拽卡片改变各服务显示顺序。
  - 如果某个服务暂时不想用，可以在设置中关闭。
- 窗口行为
  - 可选「失焦自动关闭」：用于只做一次快速翻译时，点到别处窗口后自动关闭。
  - 「置顶」按钮（小图钉）：让翻译窗口始终保持在所有窗口上方。
  - 记住窗口位置和大小：下次打开时回到你习惯的形态。

## 6. 识别窗口 – 图片文字提取

识别窗口对应「Recognize」窗口，主要为 OCR 服务。

- 左侧：图片区域
  - 展示刚刚截图得到的图片。
  - 有的版本支持重新选择或放大/缩小查看。
- 右侧：文本区域
  - 显示从图片中识别出的文字。
  - 支持复制全部识别结果。
- 控制区域
  - 选择不同 OCR 服务（例如系统 OCR、百度 OCR、腾讯 OCR、Tesseract 等）。
  - 重新识别、复制结果、直接发送到翻译窗口等操作。

## 7. 收藏 / 单词本集成（Anki / 欧路词典等）

- 功能说明
  - 将翻译结果快速添加到外部单词本或词典软件，例如：
    - Anki
    - 欧路词典（Eudic）
- 使用方式
  - 在翻译结果卡片上，通常会有「添加到 Anki」「添加到欧路」等按钮。
  - 首次使用前，需要在「设置 → 服务 → Collection」中完成账号或路径设置。
  - 配置好后，点击一次即可将当前词汇或句子作为卡片/生词添加到对应软件。
- 适用场景
  - 边查词边记单词，用于语言学习。

## 8. 朗读（TTS 语音）

- 功能说明
  - 对翻译结果或原文进行语音朗读，帮助练习发音或听力。
- 使用方式
  - 在翻译结果区域，通常会有一个「播放」或「喇叭」图标按钮。
  - 点击后，会调用 TTS 服务（例如 Lingva TTS）播放语音。
  - 可在「设置 → 服务 → TTS」中配置和选择具体的语音服务。
- 适用场景
  - 跟读句子，练习语音语调。
  - 听译内容，辅助听力练习。

## 9. 历史记录

- 功能说明
  - 自动保存你的翻译和识别记录，方便后续回查。
- 查看方式
  - 在「设置 → 历史」页面查看。
  - 可以按时间、来源等维度浏览和搜索。
- 典型操作
  - 快速找到之前翻译过的一段话。
  - 再次复制、收藏或发到单词本。

## 10. 设置中心

设置窗口是 Pot 的「控制面板」，包含多个类别。

### 10.1 通用设置

- 应用语言：界面使用的语言（简体中文、繁体中文、英文等）。
- 主题风格：浅色/深色/跟随系统。
- 字体与字号：整体界面字体及大小，适配不同分辨率和阅读习惯。
- 开机自启：是否随系统启动。

### 10.2 服务管理

- 翻译服务
  - 开启/关闭不同翻译引擎。
  - 调整显示顺序：影响翻译窗口中的卡片排列。
  - 配置各服务的 API Key、域名等参数。
- OCR 识别服务
  - 选择启用的 OCR 服务类型（系统 OCR、百度、腾讯、Tesseract 等）。
  - 对应服务的账号 / Key 配置。
- TTS 语音服务
  - 启用的 TTS 服务，语言/声音等配置。
- 收藏服务
  - 配置 Anki、欧路词典等外部工具的连接方式。
- 插件管理
  - 支持通过插件扩展新的翻译、识别、TTS、收藏服务。
  - 安装插件后会出现在对应服务列表中，可以像内置服务一样管理。

### 10.3 快捷键

- 常见快捷键类型
  - 划词翻译
  - 输入翻译
  - 截图识别
  - 截图翻译
- 特点
  - 支持自定义组合键。
  - 若某个快捷键与系统或其他软件冲突，Pot 会提示无法注册，你可以调整为其他组合。

### 10.4 备份与同步

- 功能说明
  - 将 Pot 的配置（包括服务设置、快捷键、历史记录等）备份到本地或云端。
- 支持的方式
  - 本地文件备份：导出为本地文件，重装或更换设备时可以导入。
  - WebDAV、阿里云等：通过网络存储远程备份。
- 建议使用方式
  - 在配置好大量服务和快捷键后，进行一次完整备份。
  - 在新设备上安装 Pot 后，直接导入备份，实现「一键搬家」。

### 10.5 关于与更新

- 关于页面
  - 显示当前版本号、开源地址、日志目录、配置目录等信息。
  - 提供问题反馈、加入社区等入口。
- 更新机制
  - Pot 会自动检测新版本，在后台提示你升级。
  - 也可以手动通过「检查更新」或打开更新窗口查看详细更新内容。

## 11. 典型使用场景一览

- 日常阅读外文网页
  - 划词翻译：遇到不懂的句子立即翻译。
  - 截图翻译：对无法选中文字的图片或 PDF 页面进行翻译。
- 语言学习
  - 翻译 + 收藏：查词后一键加入 Anki 或欧路词典。
  - 朗读：听标准发音，配合文本练习跟读。
- 办公/资料整理
  - OCR 识别：从扫描件、照片中提取文字，方便编辑与归档。
  - 历史记录：快速回顾最近翻译/识别过的内容。
